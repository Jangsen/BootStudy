3. HTTP 메소드

HTTP는 클라이언트의 다양한 요청을 메소드를 통해 서버로 보내는 역할을 함.

** 대표적인 메소드 **

POST		데이터 생성 요청

GET		데이터 조회 요청

PATCH(PUT)	데이터 수정 요청

DELETE		데이터 삭제 요청

4. UPDATE 문

데이터를 수정할 때 사용하는 SQL 문

형식) UPDATE 테이블명 SET 속성명 = '변경할 값' WHERE 조건;
